**Self-learning memory for AI agents.** One daemon per project, one knowledge
graph. Recall = sub-millisecond graph walk. No LLM call at read time, no cloud.

```
MCP ingest ─────────────────────► typed claims ─┐
.kern/intake/ drop → distill (LLM) → typed claims ┴→ graph → recall
```

No automatic capture — writes are always the caller's call.
